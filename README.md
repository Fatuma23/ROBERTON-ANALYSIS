# Roberton Dataset Analysis 📊

## Overview
This project analyzes the **Roberton Dataset** using Python.  
The dataset contains information on 100 volunteers, including:
- Age
- Gender
- Handedness (RH/LH)
- Physical Activity levels
- Reaction Time (numeric)
- Reaction Time (Quick/Slow category)

The goal is to explore relationships between demographic/lifestyle factors and reaction times.

---

## Tools & Libraries
- **Python 3**
- **Pandas** → data handling
- **Matplotlib & Seaborn** → visualization
- **NumPy** → numerical operations
